# 🍱 Kochi Kosen Shokudou Menu
> **Automated Intelligent Cafeteria Menu Management System (AI-Powered)**

A web application that helps students and staff easily access the cafeteria menu of **Kochi NIT (Kosen)**. The system automatically fetches PDF menu files from the school's website and uses AI to extract and display the data in a beautiful and modern interface.

---

## 🌟 Key Features

*   **🔄 Automated Sync**: The system automatically checks and fetches the latest menu PDF from the school website based on the current week.
*   **🤖 AI Data Extraction**: Utilizes **Google Gemini (2.5 Flash / 2.0 Flash)** to read and extract Japanese menu data from PDF/images, converting it into accurate digital data.
*   **📱 Modern & Responsive UI**: The menu page is designed with a premium Glassmorphism style, supporting all screen sizes (Mobile-First).
*   **🖼️ Dynamic AI Food Previews**: Displays simulated food menu images generated by AI to enhance visual appeal.
*   **⚙️ Powerful Admin Panel**: A back-office system where you can upload files manually, edit data, or delete records to trigger a re-sync.
*   **💾 Multi-Database Support**: Supports both **SQLite** (for local/personal use) and **PostgreSQL** (for production deployment).

---

## 🛠️ Tech Stack

*   **Backend**: Python 3.12+ (Flask Framework)
*   **AI Engine**: Google Gemini API (Multimodal v1beta)
*   **Database**: SQLite (Local) / PostgreSQL (Production)
*   **Frontend**: Vanilla HTML5, Modern CSS, JavaScript (ES6)
*   **Deployment**: Render.com / Gunicorn WSGI

## 🚀 Getting Started (Local Installation)

1.  **Clone Repository**
    ```bash
    git clone https://github.com/Kunpacito/KochiKosen_menu.git
    cd KochiKosen_menu
    ```

2.  **Install Required Libraries**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Setup Environment Variables**
    *   Create a `.env` file in the root directory.
    *   Add your API Key:
    ```env
    GEMINI_API_KEY=AIzaSy... (Your key from Google AI Studio)
    ```

4.  **Run Application**
    ```bash
    python app.py
    ```
    Access at: `http://127.0.0.1:5000`

---

## 🌐 Deployment

This project is pre-configured for **Render.com**:

1.  **GitHub**: Push the code to your private repository.
2.  **Render Setup**: Create a new "Web Service" and select this repo.
3.  **Environment**: Add a variable named `GEMINI_API_KEY` in the Settings page.
4.  **Database (Optional)**: For data durability, connect to an external PostgreSQL by adding the `DATABASE_URL` variable. (If not provided, the system defaults to SQLite).
